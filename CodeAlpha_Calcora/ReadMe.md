# Calcora

Calcora is a modern web-based calculator built using HTML, CSS, and JavaScript. It provides a clean and intuitive user experience with standard and scientific calculations, calculation history, keyboard support, theme switching, and local storage integration.

# Features

* Basic arithmetic operations: Addition, Subtraction, Multiplication, and Division
* Percentage calculations with accurate handling
* Scientific functions including Square Root, Square, Sine, Cosine, Tangent, Logarithm, π, and e
* Interactive calculation history
* Click-to-reuse previous results
* Dark and light theme switching
* Keyboard input support
* Responsive user interface
* LocalStorage-based theme and history persistence

# Technology Stack

Frontend: HTML5, CSS3, JavaScript (ES6)
Storage: LocalStorage
Design: Glassmorphism UI
Icons: Unicode Symbols

# Getting Started

1. Open the project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Run `index.html` using Live Server.
