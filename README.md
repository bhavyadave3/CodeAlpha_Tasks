# CodeAlpha_Tasks

This repository contains three web development projects completed as part of the internship program offered by @CodeAlpha. Each project focuses on creating modern, user-friendly, and responsive web applications using HTML, CSS, and JavaScript.

Project 1: Audique

Audique is a modern music player designed to deliver an engaging audio experience through an intuitive interface and powerful playlist management features.

Features:

* Play, pause, next, and previous controls
* Playlist creation and management
* Favorites and recently played tracks
* MP3 file upload support
* Song search functionality
* Shuffle and repeat modes
* Dark and light theme switching
* Local storage integration for persistent data
* Responsive design for desktop and mobile devices

Technologies:

Frontend: HTML, CSS, JavaScript
Storage: Local Storage


Project 2: Calcora

Calcora is a modern calculator application that combines standard and scientific calculations with a clean user interface and productivity-focused features.

Features:

* Basic arithmetic operations
* Scientific calculator functions
* Percentage calculations
* Calculation history tracking
* Keyboard support
* Dark and light theme switching
* Local storage support
* Responsive design
* Clean and modern user interface

Technologies:

Frontend: HTML, CSS, JavaScript
Storage: Local Storage


Project 3: GalleryX

GalleryX is a modern image gallery application that provides an elegant platform for viewing, organizing, and managing images with a smooth user experience.

Features

* Image upload functionality
* Gallery view with responsive layout
* Search and filtering options
* Image preview and management
* Dark and light theme support
* Local storage integration
* Mobile-friendly design
* Modern and intuitive interface

Technologies:

Frontend: HTML, CSS, JavaScript
Storage: Local Storage
