# GalleryX

GalleryX is a modern web-based media gallery built using HTML, CSS, and JavaScript. It provides an intuitive and responsive experience for managing images, GIFs, and videos with media categorization, search functionality, favorites management, drag-and-drop uploads, theme switching, and interactive viewing capabilities.

# Features

* Upload and manage Images, GIFs, and Videos
* Automatic media categorization into dedicated sections
* Responsive gallery layout for desktop, tablet, and mobile devices
* Drag-and-drop media upload support
* Lightbox image and GIF viewer
* Dedicated video playback interface
* Search functionality for quick media discovery
* Favorites management system
* Trash section for deleted media
* Dark and light theme switching
* Context menu actions for media management
* Mobile-friendly sidebar navigation
* Interactive toast notifications
* Modern and intuitive user interface

# Technology Stack

Frontend: HTML5, CSS3, JavaScript (ES6)

Storage: Browser Memory (Client-Side)

Design: Modern Responsive UI

Icons: Font Awesome

# Getting Started

1. Open the project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Run `index.html` using Live Server.
4. Upload images, GIFs, or videos to start using the gallery.
