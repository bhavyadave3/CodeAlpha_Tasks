# Audique

Audique is a modern music player built using HTML, CSS, and JavaScript. It delivers an intuitive music listening experience with playlist management, favorites, recently played tracks, MP3 uploads, theme switching, and local storage support.

# Features

* Play, pause, next, and previous controls
* Song search functionality
* Playlist creation and management
* Favorites and recently played sections
* MP3 file upload support
* Shuffle and repeat modes
* Dark and light theme switching
* Responsive vinyl-inspired interface
* LocalStorage-based data persistence

# Technology Stack

Frontend: HTML5, CSS3, JavaScript (ES6)
Storage: LocalStorage
Audio: HTML Audio API
Icons: Font Awesome

# Getting Started

1. Open the project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Run `index.html` using Live Server.
